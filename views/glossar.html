개발용어 코드 단어집

디지털 정보화 시대의 핵심 프로그래밍 언어와 기술적 개념을 체계적으로 정리한 마스터 데이터베이스입니다. 효율적인 학습과 참조를 위한 정밀한 정의와 코드 예제를 제공합니다.

클로저

Closure

함수와 그 함수가 선언된 렉시컬 환경과의 조합입니다. 외부 함수의 생명 주기가 끝나도 내부 함수가 외부 함수의 변수에 접근할 수 있게 합니다.

function outer() {
  let count = 0;
  return () => ++count;
}
TYPE: DEVOPS

도커

Docker

애플리케이션을 신속하게 구축, 테스트 및 배포할 수 있는 소프트웨어 플랫폼입니다. 소프트웨어를 컨테이너라는 표준화된 유닛으로 패키징합니다.

$ docker build -t app .
$ docker run -p 80:80 app
TYPE: FRAMEWORK

리액트

React

사용자 인터페이스를 만들기 위한 선언적이고 효율적이며 유연한 JavaScript 라이브러리입니다. 컴포넌트 기반 아키텍처를 따릅니다.

const App = () => {
  return <div>Hello World</div>;
}
TYPE: NETWORK

REST API

Representational State Transfer

자원을 이름으로 구분하여 해당 자원의 상태를 주고받는 모든 것을 의미합니다. HTTP 프로토콜의 장점을 활용하는 아키텍처 스타일입니다.

GET /api/v1/users
POST /api/v1/posts
Content-Type: application/json
TYPE: ALGORITHM

재귀

Recursion

어떤 문제를 해결하기 위해 자신을 다시 호출하여 문제를 더 작은 단위로 쪼개어 해결하는 프로그래밍 기법입니다.

function factorial(n) {
  if (n === 0) return 1;
  return n * factorial(n - 1);
}
TYPE: JS_CONCEPT

호이스팅

Hoisting

인터프리터가 변수와 함수의 메모리 공간을 선언 전에 미리 할당하는 것을 의미합니다. 코드 최상단으로 끌어올려진 것처럼 동작합니다.

console.log(name); // undefined
var name = "Terminal";
TYPE: JS_CONCEPT

비동기 처리

Async / Await

비동기 코드를 마치 동기 코드처럼 작성할 수 있게 해주는 문법적 설탕(Syntactic sugar)입니다. 가독성을 대폭 향상시킵니다.

async function fetch() {
  const data = await api.get();
  return data;
}
TYPE: ARCHITECTURE

마이크로서비스

Microservices (MSA)

단일 애플리케이션을 작고 독립적으로 배포 가능한 서비스들의 집합으로 설계하는 방식입니다. 서비스 간 통신은 주로 HTTP/REST를 사용합니다.

AuthService -> Port 8001
UserService -> Port 8002
OrderService -> Port 8003
TYPE: SECURITY

SQL 주입

SQL Injection

악의적인 SQL 문을 데이터베이스에 주입하여 데이터를 조작하거나 탈취하는 공격 방식입니다. 입력값 검증이 필수적입니다.

SELECT * FROM users 
WHERE id = '' OR '1'='1';
