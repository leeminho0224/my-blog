코딩의 역사

에이다 러브레이스의 초기 해석기관 알고리즘부터 현대의 분산형 프로그래밍까지, 코드가 세상을 정의하는 방식을 탐구합니다.

// TIME_COMPLEXITY
O(∞)
Era 0x01 // 19th Century

The Prophet of Logic

"해석기관은 직물을 짜듯이 대수학적 패턴을 짜낸다."
- 에이다 러브레이스

최초의 프로그래머, 에이다 러브레이스

찰스 배비지의 해석기관을 위해 작성된 베르누이 수 계산 알고리즘은 인류 역사상 최초의 '컴퓨터 프로그램'으로 기록됩니다. 그녀는 기계가 단순한 계산을 넘어 예술과 음악까지 창조할 수 있다는 선견지명을 가졌습니다.

Protocol // FORTRAN

과학 계산의 탄생

1957년, 고수준 언어의 효시인 FORTRAN이 등장하며 복잡한 기계어 대신 인간이 이해할 수 있는 수학적 표현으로 코딩이 가능해졌습니다.

Protocol // COBOL

비즈니스 로직의 표준

그레이스 호퍼 박사가 주도한 COBOL은 비즈니스 데이터 처리를 위한 표준을 정립했으며, 현재까지도 전 세계 금융 인프라를 지탱하고 있습니다.

Era 0x02 // 1950s - 1960s

The Birth of Syntax

천공 카드를 넘어서 정형화된 프로그래밍 문법이 탄생한 시기입니다. 폰 노이만 구조 위에서 추상화된 논리가 비로소 언어의 형태를 갖추기 시작했습니다.

Era 0x03 // 1970s - 1980s

The C Revolution

데니스 리치와 켄 톰슨에 의해 탄생한 C 언어와 UNIX 시스템은 현대 운영체제와 프로그래밍 패러다임의 기초가 되었습니다.

terminal
$ cat core_logic.c

printf("Hello, Digital World!\n");
return 0;

STRUCTURE

포인터와 직접 메모리 관리를 통해 하드웨어의 통제권을 프로그래머에게 부여했습니다.

PORTABILITY

서로 다른 아키텍처 사이에서 코드를 재사용할 수 있는 이식성의 시대를 열었습니다.

LEGACY

오늘날의 C++, Java, JavaScript까지 이어지는 중괄호({}) 문법의 시조입니다.

Era 0x04 // 1990s - Present

The Democratization of Code

인터넷의 보급과 함께 Python, JavaScript, Ruby 같은 고수준 스크립트 언어들이 폭발적으로 성장했습니다. 이제 코딩은 전문가의 영역을 넘어 창의적인 표현의 수단이 되었습니다.
